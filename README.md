# weather-app

## What is included

- React
- Redux
- Redux-thunk
- Jest
- Scss
- [Weatherbit public api](https://www.weatherbit.io/)

## Run

`yarn start`

## Build
`yarn build`

## Test

test component: `yarn test:component`

test store: `yarn test:store`

test all: `yarn test`

## Configuration
create a .env file at the root level

REACT_APP_API_KEY = <Your Weatherbit API key>
